# 🏥 MediTrack Sync 

MediTrack Sync is a **Java Swing + MySQL based desktop application** designed to manage hospital operations efficiently. It helps in managing patients, doctors, and appointments in a simple and structured way.

---

## ✨ Features

### 🧑‍⚕️ Patient Module
- ➕ Add Patient  
- 👁️ View Patient  
- 🔍 Search Patient  
- ❌ Delete Patient  

### 👨‍⚕️ Doctor Module
- 👁️ View Doctor  
- 🔍 Search Doctor  

### 📅 Appointment Module
- 📌 Set Appointment  
- 📋 Get Appointment Details  

---

## 🛠️ Tech Stack

- Java (Swing – NetBeans GUI)
- MySQL Database
- JDBC (Java Database Connectivity)
- NetBeans IDE

---

## 🗃️ Database

**Database Name:** Hospital

### Tables Used:
- doctor  
- patient  
- appointment 
---

## 🚀 How to Run the Project

1. Clone or download the project  
2. Open it in **NetBeans IDE**  
3. Create MySQL database: `Hospital`  
4. Import or create required tables  
5. Update MySQL username/password in code  
6. Run `Main.java`

---

## 🎯 Project Objective

The main goal of this project is to digitize basic hospital management tasks such as:
- Managing patient records  
- Managing doctor details  
- Handling appointments efficiently
